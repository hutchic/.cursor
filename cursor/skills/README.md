# Skills

This directory contains Cursor automation skills.

## Available Skills

### `command-development` - Command Development Guidelines

Provides AI context and best practices for developing Cursor IDE commands following repository standards.

**Topics covered:**
- Structure decision matrix (file vs directory)
- Required elements for all commands
- Integration patterns
- Testing requirements
- Common code patterns
- Repository-specific conventions

[View Skill](command-development.md)

## What are Skills?

Skills are knowledge files that provide context to Cursor AI:
- **Format**: Markdown files
- **Purpose**: AI reference material, not executable code
- **Examples**: Code style guides, architecture patterns, development guidelines

## Structure

Place skill files here for symlink installation to `~/.cursor/skills`.

## Creating New Skills

Skills should:
1. Be written in Markdown
2. Provide clear, actionable guidance
3. Include examples where appropriate
4. Focus on a specific domain or practice
5. Be concise but comprehensive

See [COMMAND_DEVELOPMENT_GUIDE.md](../COMMAND_DEVELOPMENT_GUIDE.md) for detailed information about developing commands and skills.
